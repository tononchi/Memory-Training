# 学年帯別「読みの伸ばし方」― 文献調査と方針判断

本資料は、「低学年では“速く読むだけ”に留まらない指導が良い」という[既存の調査](speed-reading-research.md)を踏まえ、**中学年・高学年・中学生・高校生**では何が有効かをメタアナリシス中心に整理し、本リポジトリのアプリ設計方針を判断するためのものです。

> **方針の結論（先に）**
> - **反復読み（流暢性／自動化）型の本アプリの手法は、おおむね小学6年（高学年）まで有効**（ただし学年が上がるほど効果量は逓減）。→ **問題データを3〜6年まで追加**する。
> - **中学生以降は「別の速読・読解訓練」が有効な帯**。自動化はほぼ完成し、流暢性ドリルの上乗せは小さくなる。代わりに**読解方略・語彙/形態素・多読・目的に応じた読速調整（スキミング/スキャニング）**が効く。いわゆる“高速提示型の速読”は速度こそ上がるが詳細理解は上がらない。→ **中高生向けの別アプリ**を作成する。
> - 最終更新：2026-06-29。効果量は引用元の二次記述に依拠する箇所があり、学術用途では原典確認を推奨。

## 0. 判断の根拠を一目で（効果量サマリ）

| 手法 | 主な対象 | 効果量（代表値） | 出典 |
|---|---|---|---|
| 反復読み（流暢性） | 小学校（とくに低〜中学年） | 練習文で流暢性 **d≈0.83**／理解 **0.67** | [Therrien 2004](#ref-therrien) |
| 流暢性介入（全体） | 小1〜中等 | 全体 **ES≈0.43**、学年が上ほど縮小 | [Stevens/Walczyk 系](#ref-fluencymeta) |
| 流暢性介入（4〜12年） | 上位学年・中等 | 流暢性 **+0.16**、標準化読解 **0.09〜0.13** | [Wanzek/上位学年レビュー](#ref-upperelem) |
| 音読流暢性（規準） | 1〜6年 | WCPMは**5年まで急伸→6年で頭打ち**(+約4語/分) | [Hasbrouck & Tindal 2017](#ref-orfnorms) |
| 中等の読み介入（全体） | 4〜12年 | **0.95(2007)→0.49(2015)** に縮小、標準化0.21 | [Scammacca 2007/2015](#ref-scammacca) |
| 読解方略指導 | 中・高 | 標準化 **d≈0.19**／自作テスト **0.43**／方略運用 **0.79** | [Okkinga 2018](#ref-okkinga) |
| 読解介入（3〜12年） | 苦戦読者 | **g≈0.59**（方略指導でより高い） | [Filderman 2022](#ref-filderman) |
| 形態素指導 | 学齢全般 | 読み書き **d≈0.33**、読解 **0.24**（総合指導の一部で有効） | [Goodwin & Ahn / Bowers ほか](#ref-morph) |
| 多読（Extensive Reading） | 中等・L2 | 読解 **d≈0.5〜0.9**（構造化されたERが有効） | [Nakanishi 2015 / Krashen / Kim](#ref-er) |
| 自由黙読（ISR、無構造） | 全般 | **g≈0.08**（読解は **−0.14**：効果ほぼ無し） | [Bus 2024 / Erbeli & Rice 2022](#ref-isr) |
| “速読”トレーニング | 成人・中高 | 速度は上がるが**理解は向上せず**（gist向け） | [Klimovich 2023 / Rayner 2016](#ref-speed) |

要点：**流暢性ドリルの効果は学年が上がるほど小さくなり、6年あたりで自動化（音読速度）が頭打ちになる。**それ以降は「速く読む」こと自体ではなく、**理解方略・語彙・多読・目的に応じた読速調整**が伸びしろの中心になる。

## 1. 低学年（小1〜2）― 既存方針の再確認

[speed-reading-research.md](speed-reading-research.md) のとおり、低学年は**反復読み（Repeated Reading）＋お手本提示＋即時FB＋理解確認**が中核。NRP(2000)、Therrien(2004, 流暢性d≈0.83)、Lee&Yoon(2017)。
→ 本アプリ（Memory-Training）の現行設計が該当。

---

## 2. 中学年（小3〜4）― 反復読みはまだ有効、ただし逓減

<a id="ref-therrien"></a>
**Therrien (2004) メタ分析**：反復読みは練習文で**流暢性 d≈0.83／理解 0.67**。健常児・LD児いずれにも有効で、初等段階で確実な効果。

<a id="ref-fluencymeta"></a>
**流暢性介入メタ分析（複数）**：全体の重み付き**ES≈0.43**。重要なモデレータとして**学年（若い学年ほど効果大）**と、**教材の難易度設定**が挙げられる。とくに「正確性基準で instructional level を設定」した介入は**ES≈1.03**と大きく、「速度基準」設定の**0.29**を上回る。→ **速さ基準で煽るより、正確に読める教材の反復**が効く（本アプリの「速度非強制・自己ベスト」方針と一致）。

<a id="ref-upperelem"></a>
**上位学年（4年〜）レビュー/メタ**：流暢性・語認識で**+0.16**、標準化された基礎スキル**0.09**・読解**0.13**と、低学年より明確に小さい。
→ 中学年は反復読みが**まだ十分有効**だが、効果量は低学年より下がる帯。本アプリの手法をそのまま適用可能。

---

## 3. 高学年（小5〜6）― 自動化が頭打ちに近づく「移行帯」

<a id="ref-orfnorms"></a>
**Hasbrouck & Tindal (2017) ORF全国規準**：音読語数（WCPM）は**1〜5年で急速に伸び、6年で頭打ち**（年間の伸びが平均約4語/分まで縮小、低下する百分位も）。典型児では**音読の自動化が5〜6年でほぼ完成**することを示す。
→ つまり高学年では「流暢性そのもの」の伸びしろが小さくなる。反復読みは**まだ意味があるが**、上乗せは限定的。読解・語彙へ軸足を移す準備段階。

**判断**：反復読み型の本アプリは**6年まで**は方法として妥当（自動化の総仕上げ＋理解確認）。**よって本アプリのデータを3〜6年まで拡張**する。ただしUI上のメッセージは引き続き「速さの強制」をせず、正確さ・理解・自己ベストを重視する。

---

## 4. 中学生・高校生 ― 「別の手法」が有効になる帯

ここで効果の構造が変わる。**自動化はほぼ完成しており、流暢性ドリルの上乗せは小さい**。メタ分析が支持する伸ばし方は次の通り。

<a id="ref-scammacca"></a>
**Scammacca ほか（2007→2015）中等(4〜12年)介入メタ分析**：全体効果は**0.95(2007)→0.49(2015)**、標準化指標では**0.21**に縮小。介入の主対象は**語彙・読解方略・テキスト流暢性**。流暢性単独より、語彙・方略を含む包括的指導が中心。

<a id="ref-okkinga"></a>
**Okkinga ほか (2018) 読解方略指導メタ分析（教室規模）**：標準化テスト **d≈0.19**、自作テスト **0.43**、方略運用能力 **0.79**。予測・要約・自己質問・モニタリング等の**明示的方略指導**が効く（介入は長ければ良いわけではない）。

<a id="ref-filderman"></a>
**Filderman ほか (2022) 読解介入メタ分析（3〜12年・苦戦読者, 64研究）**：読解への**g≈0.59**。**方略指導**を含む介入で効果が有意に高い。

<a id="ref-morph"></a>
**形態素・語彙介入メタ分析**：形態素指導は読み書き**d≈0.33**（読解**0.24**、語彙**0.34〜0.40**）。**単独より総合指導の一部**として、また**10〜20時間以上**で有効。中高では未知語の語構成（接辞・語根）理解が読解効率に効く。

<a id="ref-er"></a>
**多読（Extensive Reading）メタ分析**：Nakanishi(2015, TESOL Q)、Krashen、Kim(2012, d≈0.62)など、読解に**d≈0.5〜0.9**。**構造化された多読**（適切な難度選定＋アカウンタビリティ）が効く。

<a id="ref-isr"></a>
**自由黙読（無構造のISR）**：Bus ほか(2024, 47実験)で**g≈0.08**（読解は**−0.14**）、Erbeli & Rice(2022)も効果を支持せず。→ **ただ黙って読ませるだけでは伸びない**。構造（目的・選書・確認）が要る。

<a id="ref-speed"></a>
**“速読”トレーニングそのもの**：Klimovich ほか(2023)は、速読訓練・メタ認知訓練で**読む速度は上がるが理解は向上せず**、初期の語認識にも影響なしと報告。Rayner ほか(2016, PSPI)は、**詳細理解を保ったまま劇的な高速化は不可能**で、眼球運動の高速化やRSVPでは理解が落ちると結論。読速メタ分析（Brysbaert 2019）では成人の黙読は概ね**約238語/分（ノンフィクション）**。
→ いわゆる速読は**gist（要旨）取得＝スキミング**には有用だが、**精読の理解は上げない**。

**スキミング／スキャニング（目的に応じた読速調整）**：目的に合わせて速度を変える「**読みの柔軟性**」は効率を高める。スキミング＝要旨把握、スキャニング＝特定情報の探索。中高生（とくに苦戦読者）には、**プレビュー・要約・モニタリング**等の方略を明示的に教えると有効。

**結論（中高生の“別の手法”）**：RSVP的な高速強制ではなく、
1. **目的に応じた読速の使い分け**（ざっくり読み＝スキミング／さがし読み＝スキャニング／じっくり読み＝精読）、
2. **読解方略の明示的練習**（予測・要約・自己質問・見出し活用）、
3. **理解を担保したうえでの時間内読**（速度×理解の両表示、理解が落ちたら速さは評価しない）、
4. 背景として**語彙/形態素**と**構造化された多読**。
これらが中高生に適した「速読・読解トレーニング」である。

## 5. 日本の文脈（参考）

国内の速読サービス（日本速読解力協会、各塾）は小〜高校生向けに「読書速度×正答率」を併記評価する設計を採る例が多く、**速さだけでなく理解（正答率）を併測する**点は上記エビデンスと整合的。一方、学術的メタ分析として日本語の速読効果を直接示すものは乏しく、英語圏の知見の援用が中心。文科省資料でも読書能力は段階的発達とされ、無構造の読書より**目的をもった読書活動**が重視される。

## 6. 実装方針（このリポジトリへの落とし込み）

| 学年帯 | 有効な手法 | 実装 |
|---|---|---|
| 低学年(1-2) | 反復読み＋お手本＋理解確認 | 既存アプリ（実装済み） |
| 中学年(3-4) | 反復読み（逓減）＋理解確認 | **既存アプリにデータ追加**（3・4年 かんじ入り） |
| 高学年(5-6) | 反復読み（自動化の総仕上げ）＋理解 | **既存アプリにデータ追加**（5・6年 かんじ入り） |
| 中学生・高校生 | 読速の使い分け＋読解方略＋理解担保＋（語彙/多読） | **別アプリを新規作成**（中高生むけ 速読・読解） |

- 既存アプリのデータ拡張は、漢字を[学年別漢字配当表](kanji-by-grade.md)に沿わせる（各学年の累積漢字のみ使用）。
- 別アプリ（中高生むけ）は、**スキミング／スキャニング／精読の3モード＋理解チェック＋（速度×理解の併記）**を中核に、**理解が一定以上のときだけ速度を評価**する設計とし、RSVP型の高速強制は採用しない（§4の知見）。
