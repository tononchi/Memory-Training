# 低学年向け速読訓練法（調査メモ）

本資料は、小学校1〜2年生の「読む速さ」向上を目的に、教育実践・研究で支持される方法を整理したものです。  
※この実行環境では外部サイトのDNS解決ができず、Web取得ツールでのライブ取得が不可だったため、公開済みの代表的研究・ガイドライン知見（2024年までの一般的知識）を要約しています。実運用前に学校・自治体要件に合わせて最新論文の再確認を推奨します。

## 要点（結論）

- 低学年では「速く読む」単独より、**正確性 + 音読流暢性 + 理解**の同時改善が重要。  
- 有効性が高いのは、**短文の反復読み（Repeated Reading）**、**教師/音声モデル提示（Modeling）**、**即時フィードバック**、**短時間の継続練習**。  
- モチベーション維持には、**即時スコア**、**連続日数（streak）**、**バッジ/スター**などのゲーミフィケーションが有効。

## 速読訓練アプリに反映すべき設計

1. **1セッションを短く（約1分）**
   - 認知負荷を下げ、毎日継続しやすくする。
2. **短文を次々表示してテンポ練習**
   - まずは語彙・文パターンの自動化を促進。
3. **「よめた」自己申告 + 即時得点**
   - 成功体験を即時可視化。
4. **連続学習インセンティブ**
   - 連続日数・スター・バッジで習慣化を促進。
5. **理解を落とさない運用**
   - 教室運用時は別途、内容確認問題（口頭/選択）を追加する。

## 代表的な根拠（要再確認の参考文献）

1. National Reading Panel (2000). *Teaching Children to Read*  
   - 流暢性指導（特に反復読み）と読解への寄与を報告。
2. Rasinski, T. V.（複数研究）  
   - 読みの流暢性（accuracy, rate, prosody）の重要性を提唱。
3. What Works Clearinghouse（IES）Practice Guides  
   - 初等読解指導で、デコード・流暢性・理解の統合指導を推奨。
4. 近年のメタ分析（2019〜2024の読解/流暢性介入研究）  
   - 継続的・短時間・反復型の介入が初等段階で有効という傾向。
